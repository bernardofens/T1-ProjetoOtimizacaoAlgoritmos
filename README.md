# T1-ProjetoOtimizacaoAlgoritmos
Trabalho 1 da disciplina de Projeto e Otimização de Algoritmos.

# Descrição resumida do trabalho:
Implementar e analisar o algoritmo de Maior Subsequência Comum (LCS) usando programação dinâmica. Comparar com a versão recursiva de força bruta (opcionalmente). Entregar relatório com explicação do problema, exemplos, descrição dos algoritmos, análise de iterações/tempo e a implementação (código fonte organizado).
